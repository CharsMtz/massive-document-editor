Generador de documentos masivos

Selecciona el archivo Excel
Selecciona el archivo Word o PP

  1. Crea tu tabla en Excel con la información acomodada por columnas. La cabecera de cada columna será el nombre que se sustituirá dentro de tu archivo plantilla.
  2. Crea tu archivo plantilla. Dentro de este archivo sustituye las partes que quieras cambiar por los datos de la tabla de Excel. Encierra dichas palabras con corchetes {}, por ejemplo {Nombre}.
  3. Selecciona los archivos en sus respectivas entradas.
  4. Clickea el boton "Generar" para crear todos los archivos.
